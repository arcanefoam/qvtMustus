QVT Rose UML Importer Changes

Formerly the UMLX Rose UML Importer.

QVT Rose UML 0.7.0

Promoted from GMT/UMLX to M2M.

UMLX Rose UML 1.2.0 (8-Jun-2008)

Generation of traditionalemof (no PrimitiveTypes package) added.
Unnavigable opposite role names modelled using EMOF comments/Ecore annotations eliminating the need for max/min models.
Unity upperbound features forced to be ordered.
EMOF namespace changed to http://schema.omg.org/spec/MOF/2.0/emof.xml.

UMLX Rose UML 1.1.0 (24-Sep-2007)

Documentation added.
Conversion action added to support QVT model transformation and uni-directional association variants.

UMLX Rose UML 1.0.0 (6-Sep-2007)

Original importer variant with support for ordered constraints.
