This plugin contains a slightly modified build of org.eclipse.imp.runtime 0.1.100 SVN 21814.

This plugin should be considered deprecated in that it is intended to eliminate it as soon as IMP runtime is promted to a non-Incubation status.

The exact differences may be found in org.eclipse.imp.runtime.patch.

The unpatched plugin is API compliant and so may be used instead, however the following problems may be observed:

  • BuilderBase.java - new editors may start with an NPE
  • AnnotationHoverBase.java - multi-line annotations are not multi-line
  • ParserScheduler.java - an NPE may occur when an edited file is deleted
  • PresentationController.java - coloring ranges may be off by one
  • LanguageRegistry.java - Open With offers an IMP editor that functions poorly
  • HTMLPrinter.java - multi-line diagnostics are not multi-line
